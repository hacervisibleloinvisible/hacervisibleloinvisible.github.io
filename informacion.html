¿DE QUÉ SE TRATA EL PROYECTO?

El proyecto se trató de una recopilación de información acerca de las rutas que varias personas usaban para circular cerca del sector de las universidades Católica, Salesiana y Politécnica. Esto con el fin de identificar cuales eran las rutas más usadas y en base a eso poder sacar módulos por los cuales se generó una estación de autoreparación estática.

Aquí particularmente se registran 12 rutas las cuales podemos ver individualmente con el fin de entender como es la ruta de cada persona como cada una representa a su propietario.

¿CÓMO SE REALIZÓ?

Todo inicio con la conceptualización del proyecto donde se penso que la ruta que cada persona es única por lo que se podria comparar con una firma ya que es algo únco que cada persona tiene.

Una vez que se tuvo eso, pues se penso en como se iba a hacer para recolectar la información, entonces se diseño un folleto donde la persona que queria formar parte de del proyecto llenaba la información que posteriormente se digitalizaba y se adjuntaba a la página web.
